# Analise_BaseDados
Este projeto foi executado na plataforma  "Google Colaboratory" 
 Esta analise    foi feita sobre um 'recorte" da base de dados "AdventureWorks". De onde podemos fazer 
 uma serie "querys"(consultas, extracoes) como por exemplo a receita total, bem como criar novas colunas visando obter 
 outras informacoes relevantes para analise da base de dados 
 tambem podemos visualizar essas analises .
